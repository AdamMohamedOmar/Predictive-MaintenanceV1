# Data directory

This directory is mostly gitignored. See `.gitignore` at the repo root.

## Expected layout

```
data/
├── raw/
│   └── carOBD/              # Clone from https://github.com/eron93br/carOBD
│       ├── drive1.csv
│       ├── idle1.csv
│       └── ...
├── synthetic/               # Generated by scripts/generate_dataset.py (Week 3)
└── skoda_baseline/          # Committed to repo; recorded Week 6
    ├── skoda_healthy_YYYY-MM-DD.csv
    └── baseline_stats.json
```

## Obtaining the carOBD data

```bash
# From the repo root:
git clone https://github.com/eron93br/carOBD.git data/raw/carOBD
```

The `data/raw/` directory is gitignored. Each team member clones the data independently.

## Obtaining the Skoda baseline

The Skoda baseline is recorded Week 6 and committed to this repo (files are small, highly project-specific, and serve as ground truth for the live demo). See `docs/WEEK_06.md` Wednesday.
